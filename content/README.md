# Content

Deze map bevat de gekozen contentbasis voor de nieuwe persoonlijke website.

De volledige oude WordPress-export staat in `archive/source-wordpress-export/` en is via `.gitignore` buiten git gehouden. Gebruik die export alleen als bronmateriaal.

## Structuur

- `pages/home.nl.md`: hoofdverhaal voor de homepage.
- `pages/cv.nl.md`: rustige CV-pagina, niet prominent in de hoofdroute.
- `source-notes.md`: inhoudelijke samenvatting van oude site plus nieuwe context.
- `data/assets.json`: geselecteerde beeldbestanden en hun beoogde rol.
- `assets/`: beelden die als content voorbereid zijn voor de nieuwe website.

## Contentprincipe

De site moet niet bewijzen dat Jonathan veel technologieen kent. De site moet laten begrijpen hoe Jonathan denkt, werkt en verschil maakt.
